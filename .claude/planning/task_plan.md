# UMU Skills 开发与发布规划

## 项目目标

为 `umu-skills` 项目设计并实施一套 Skill 开发与发布机制，满足：

1. **开发阶段**：开发者在项目目录中持续新增功能时，已安装的 Skill 能自动获得新功能，便于调试。账号信息优先从项目中的 `.env` 文件读取。
2. **发布阶段**：项目发布到 GitHub 后，其他用户能通过自动化安装方式获取并安装 Skill。安装后如缺少账号信息，通过交互方式引导用户录入，信息存放在 Skills 目录中并加密保护。用户可通过自然语言指令修改账号信息。

## 当前状态

- 已配置 MCP server（`.claude/settings.json`）

- 账号信息当前通过环境变量或 `.env` 文件传递
